## §1 — لماذا كل منصة ترفع جوجل (الميكانيزم بالتفصيل)

### 🎥 YouTube — أقوى رافعة SEO على الإطلاق
- فيديوهات يوتيوب تظهر **داخل صفحة نتائج جوجل** (SERP) في خانة الفيديو وأحياناً فوق النتائج النصية.
- **embed** الفيديو داخل مقال المدونة يرفع *dwell time* و*pages/session* → إشارة جودة قوية لجوجل.
- وصف الفيديو + الـ chapters + الـ transcript = **نص قابل للفهرسة** يحمل كلماتنا المفتاحية.
- القناة نفسها تبني **backlink** من يوتيوب (DA عالي جداً) للموقع.

### 📱 TikTok — محرّك البحث المُعلَّم (Branded Search)
- شراكة TikTok مع جوجل تجعل بعض الفيديوهات **تُفهرس وتظهر في بحث جوجل**.
- TikTok نفسه أصبح **محرك بحث** للجيل الأصغر ("ازاي أعرف الشاحن الأصلي").
- الأثر الأكبر: مشاهدة المحتوى → **بحث عن "كايرو فولت" بالاسم** = أقوى إشارة ثقة لجوجل.

### 🐦 Twitter/X — الفهرسة السريعة + الكيان
- جوجل يفهرس تغريدات X **خلال دقائق** → سرعة ظهور للأخبار والروابط الجديدة.
- Threads المبنية على المقالات = **روابط ترجع للموقع** + إشارات E-E-A-T (خبرة، تخصص).
- ممتاز لإعلان كل مقال جديد فور نشره → تسريع الـ indexing.

## §3 — محرك المحتوى: ذرّيّة المحتوى (Content Atomization)

**المبدأ:** كل مقال واحد (1500+ كلمة) = **مصنع** لـ 6–8 أصول سوشيال. لا نبدأ من صفر أبداً — نعيد تدوير المنجم.

```
مقال واحد (Pillar)
   ├── 🎥 1 × YouTube Long-form (2–4 دقائق) — الشرح الكامل
   ├── 🎬 2–3 × Shorts / Reels (15–30 ثانية) — كل "نصيحة" أو "خطأ" = Short
   ├── 📱 1–2 × TikTok native — هوك + ميم + معلومة صادمة
   ├── 🖼️ 1 × Carousel صور (Nano Banana) — للإنستجرام/فيسبوك
   └── 🐦 1 × Twitter Thread — أهم 5–7 نقاط + رابط المقال
```

**مصفوفة "نوع المقال → أفضل صيغة سوشيال":**

| نوع المقال | أمثلة من المكتبة | أفضل صيغة فيديو |

### 💰 ميزانية الـ Credits (القيد الحاكم)
- الرصيد الآن: **4.5 credits** → يكفي ~1 فيديو فقط.
- الكادينس المخطط: **~12 فيديو/شهر** (انظر §7) → ميزانية شهرية مطلوبة.
- **الإجراء:** شحن باقة credits بداية كل شهر. تحقق من التكلفة الفعلية للفيديو عند أول توليد (`generate_video` يعرض التكلفة)، ثم اضبط الباقة.
- **حارس الميزانية:** لو الرصيد < تكلفة فيديوين → أوقف التوليد وأبلغ، لا تستهلك آخر credit.

  - **العنوان:** كلمة مفتاحية + سنة — "أفضل باور بانك في مصر 2026 (تجربة حقيقية)".
  - **الوصف:** أول سطرين = الكلمة المفتاحية + **رابط المقال** `https://cairovolt.com/blog/{slug}` · ثم chapters · ثم روابط منتجات.

  - **Chapters** (timestamps) + **end screen** يوديّ لمقطع تاني + **pinned comment** فيه رابط الموقع.
  - **Playlists** لكل عنقود (Power Banks / Chargers / Fake-Detection / Earbuds).
- **🔗 الأهم لـ SEO:** **embed كل long-form داخل مقال المدونة المقابل** → يرفع dwell time مباشرة.

### 🥈 TikTok (محرّك Branded Search)
- **الحساب:** @cairovolt · bio فيه "اكسسوارات موبايل أصلية - مصر" + رابط الموقع (link-in-bio).
- **الصيغة:** Shorts 9:16 فقط، هوك في أول ثانية، نص على الشاشة (لأن 80% بيتفرجوا بدون صوت).
- **SEO:** **اكتب الكلمة المفتاحية في الكابشن والنص المرئي** ("ازاي تعرف الشاحن الأصلي") — TikTok search + Google indexing بيقروهم.
- **Hashtags:** `#كايرو_فولت #شواحن #باور_بانك #مصر #انكر` + هاشتاج تريند مناسب.

### 🥉 Twitter/X (الفهرسة السريعة + E-E-A-T)
- **الحساب:** @cairovolt · bio خبير + رابط الموقع · صورة + هيدر بالعلامة.
- **الصيغ:** (أ) Thread من كل مقال (5–7 تغريدات + رابط) (ب) إعلان كل مقال جديد فور نشره (تسريع indexing) (ج) تغريدات نصائح سريعة يومية.
- **SEO:** كل thread يقفل بـ **رابط المقال** — جوجل يفهرسه بسرعة ويعطي إشارة طزاجة.

> **⚖️ قانون i18n (إلزامي):** الكابشن/الوصف **العربي** يربط `cairovolt.com/blog/{slug}` (بدون `/en/`). مسار **EN** يربط `cairovolt.com/en/blog/{slug}` فقط. خلط المسارين = خطأ.

نكرّر نفس النمط الناجح لـ FB/IG. الأصول الجديدة تُبنى على نفس البنية:

| المنصة | API | درجة الأتمتة | السكربت المقترح |
|--------|-----|---------------|------------------|
| YouTube | **Data API v3** (OAuth, free quota ~6 رفعات/يوم) | 🟢 أتمتة كاملة | `scripts/upload-to-youtube.mjs` |
| Twitter/X | **API v2** (free ~500 كتابة/شهر) | 🟢 أتمتة كاملة | `scripts/post-to-twitter.mjs` |
| TikTok | **Content Posting API** (يتطلب مراجعة تطبيق) | 🟡 شبه يدوي أول الأمر → أتمتة بعد الموافقة | `scripts/post-to-tiktok.mjs` |

**التدفق المؤتمت (مرآة لـ FB/IG):**
```
توليد الفيديو/الصورة (Seedance/Nano Banana عبر MCP) → حفظ في /public/social/{slug}/
        ↓
git push → GitHub Action جديد (social-video-post.yml)
        ↓
detect → يقرأ بيانات المقال (نفس detect-new-articles.mjs)
        ↓
ينشر: YouTube (auto) + X (auto) + TikTok (auto/يدوي)
```
**Secrets المطلوبة (GitHub):** `YT_CLIENT_ID/SECRET/REFRESH_TOKEN` · `X_API_KEY/SECRET/ACCESS_TOKEN` · `TIKTOK_ACCESS_TOKEN`.

> **ملاحظة نطاق:** هذه الخطة = **التصميم الكامل القابل للتنفيذ**. بناء السكربتات الفعلي = مرحلة تالية (أقدر أبنيها فور موافقتك). الجدولة أدناه تعمل **يدوياً أو مؤتمتاً** بنفس الكفاءة.
